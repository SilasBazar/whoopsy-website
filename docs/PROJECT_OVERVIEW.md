# WHOOPSY Website - Project Overview

## Mission

WHOOPSY is a one-person game studio driven by an obsession with creating games that have soul. This website showcases our games, philosophy, and connects players with our vision.

## Current Games

### Leap of Faith
- **Status**: Announced
- **Genre**: Party Game / Local Multiplayer
- **Platform**: PC (Steam)
- **Description**: Unlimited local multiplayer chaos! Connect infinite controllers & phones, control wobbly giraffes, dodge physics-based obstacles, and fight to reach the rocket ship first.

## Tech Stack

This website is built with modern web technologies:

- **Build Tool**: Vite 5.0
- **Framework**: React 18.2
- **Language**: TypeScript 5.2
- **Styling**: Tailwind CSS 3.3
- **Deployment**: GitHub Pages (via GitHub Actions)

## Design Philosophy

### Brand Identity
- **Colors**:
  - Green: `#86ce43` (Primary, accent)
  - Black: `#1d1c00` (Background, text)
  - White: `#e5e6cc` (Cards, buttons)
- **Fonts**:
  - Display: Londrina Solid (Headings, logo)
  - Mono: Space Mono (Body text)
- **Style**: Bold, playful, 3D effects, clean layout

### Key Design Elements
1. **3D Buttons**: Multi-layer shadow effect with spark animation on click
2. **Bold Typography**: Large display fonts with strong hierarchy
3. **Playful Interactions**: Hover effects, smooth scrolling, glitch animations
4. **Simple Containers**: Cards and stats have clean, static shadows

## Project Goals

### Primary Goals
- Showcase WHOOPSY's games and philosophy
- Provide clear call-to-action for Steam wishlist
- Reflect the brand's playful, soulful nature
- Maintain fast loading times and excellent performance

### Future Goals
- Add blog/news section
- Create individual game pages
- Add game demos/iframe integration
- Implement dynamic content management
- Add multiplayer features (chat, forums)
